# 관리자 대시보드 재설계 계획

## 변경 사항
기존 카드 형태의 관리 기능을 왼쪽 사이드바 메뉴 형태로 변경하여 가독성과 접근성을 향상시킵니다.

## 새로운 구조

### 1. 왼쪽 고정 사이드바 (w-64)
- 로고 및 대시보드 홈
- 통계 대시보드
- 사용자 관리
- 구인정보 관리
- 구직정보 관리
- 협약대학교 관리
- 에이전트 관리
- 고객 지원

### 2. 오른쪽 메인 콘텐츠 영역
- 간소화된 상단 헤더 (사용자 정보, 알림)
- 대시보드 통계 카드
- 각 관리 기능의 상세 화면

## 구현 단계
1. 전체 레이아웃을 flex 구조로 변경
2. 왼쪽 사이드바 컴포넌트 생성
3. 헤더 간소화
4. 반응형 모바일 메뉴 추가
